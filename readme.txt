*** Software Package for Computing Logical Relationships in Biological Databases ***

1. Installation
	1.1 Download the contents of this folder to a desired location on your computer.
	1.2 Open MATLAB, `cd` to the package directory downloaded above, type "setup" and then press Enter (this step adds the package directory to MATLAB search path).

2. Performing calculations
	2.1 Open "logical_relationships.m" file.
	2.2 Set the input parameters based on your case of study.
	2.3 Click on the "Run" button to perform computations.
	
3. Results
	3.1 Results will be saved as PDF files in the sub-directory "results".
