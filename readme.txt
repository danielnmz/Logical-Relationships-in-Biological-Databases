*** Software Package for Computing Logical Relationships in Biological Databases ***

1. Installation
	1.1 Copy and paste the folder "LR-2.0" to a desirable location on your computer.
	1.2 Open MATLAB and type "setup" and then press Enter (this step adds the current directory for "LR-2.0" to MATLAB search path).

2. Performing calculations
	2.1 Open "logical_relationships.m" file.
	2.2 Set the input parameters for your case of study.
	2.3 Click on the "Run" button.
	
3. Results
	3.1 Results are saved as PDF files in the subfolder "results".
